Powershell Commands

Command Description
Get-Help <cmd> Displays detailed help information about PowerShell commands (<cmd>). Use -Examples for practical examples.
Get-Command Lists all commands available in PowerShell, including cmdlets, functions, and aliases.
Get-ChildItem Lists items in the current directory. Equivalent to ls or dir in other shells.
Set-Location <path> Changes the current directory to <path>. Equivalent to cd in other shells.
New-Item <path> -ItemType File Creates a new file at the specified path.
New-Item <path> -ItemType Directory Creates a new directory at the specified path.
Copy-Item <source> <destination> Copies a file or directory from <source> to <destination>.
Move-Item <source> <destination> Moves a file or directory from <source> to <destination>.
Remove-Item <path> Deletes a file or directory at the specified <path>.
Get-Content <file> Displays the content of a file. Similar to cat in Unix/Linux.
Set-Content <file> -Value <content> Writes or replaces the content in a file with <content>.
Add-Content <file> -Value <content> Appends <content> to the end of a file.
Select-String <pattern> <file> Searches for a <pattern> in a file. Similar to grep in Unix/Linux.
Get-Process Lists running processes. Use -Name <name> to filter by process name.
Stop-Process -Name <name> Stops a process by name. Can also use -Id <id> to stop by process ID.
Get-Service Lists all system services. Use -Name <name> to filter by service name.
Start-Service -Name <name> Starts a service by name.
Stop-Service -Name <name> Stops a service by name.
Get-EventLog -LogName <logName> Retrieves events from a specified log. For example, use System or Application for <logName>.
Get-SystemInfo Displays detailed system information.
